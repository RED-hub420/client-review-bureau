Privacy Notice

Directory privacy notice

Reference notice describing public summaries, protected records, and publication limits.

Reference

Public summaries only

Public directory records show screened summary information and current file status.

Protected information

Identity numbers, raw uploads, account information, medical details, and internal notes are not displayed in the public directory.

Limited detail during review

Public detail may be reduced while a dispute is active or while supporting information is incomplete.

Correction requests

Correction requests and response statements remain attached to the same file and may result in status or visibility changes.
