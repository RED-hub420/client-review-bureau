Search notes

Public view
  • Only published file summaries appear in the directory.
  • Some names may be shortened or partially masked.
  • Files can show reduced detail while a dispute is open.
  • Not every submitted review becomes a published file.

Publication levels

Reference
  • Public summary Basic issue, status, and activity timeline are visible.
  • Limited summary Open disputes or missing support can reduce public detail.
  • Closed file Summary remains visible with final bureau status attached.

How directory status moves

Status guide
Pending review

The filing has been received and is in bureau screening or notice preparation.

Answered

A response has been attached to the file and is reflected in the public record.

Unresolved

A response was received, but the matter remains unresolved in the public summary.

Unanswered

No response was received during the current handling window.

Directory use notice

Important

Directory information is provided for marketplace reference only. A published file is not the same thing as a court finding, a government action, or a consumer credit decision.
