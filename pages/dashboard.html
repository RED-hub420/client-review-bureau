Directory Notes

Public view
  • Only published file summaries appear in the directory.
  • Certain identifying details may be limited or abbreviated.
  • File detail may be limited during active review.
  • Not all submitted reviews are accepted for publication.

File Visibility Levels

Reference
  • Public summary Basic file information, status, and activity timeline.
  • Limited summary Active review or incomplete documentation may limit detail.
  • Closed file Final status remains recorded in the directory.

How directory status moves

Status guide
Pending review

The filing has been received and is in bureau screening or notice preparation.

Answered

A response has been attached to the file and is reflected in the public record.

Unresolved

A response was received, but the matter remains unresolved in the public summary.

Unanswered

No response was received during the current handling window.
